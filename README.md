README
==================

This folder contains the source code and data for the Text Mining project. 

The clean dataset is stored as a json file as 'bookshelf.json'.
To replicate the study, please find the following two jupyter notebooks:

* [scrape.ipynb](http://nbviewer.jupyter.org/github.com/rogercusco/marx_vs_smith/scrape.ipynb) Contains the code to build the file bookshelf.json.
  
* [parse.ipynb](http://nbviewer.jupyter.org/github.com/rogercusco/marx_vs_smith/parse.ipynb) Contains the code to replicate the analysis.
